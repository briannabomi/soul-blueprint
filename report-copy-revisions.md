## Overall Tone Notes for Report

The report currently has two tonal modes fighting each other:

1. **Informational mode** — "Your Life Path is 5: The Freedom Seeker. You are here to embrace adventure..." This is clear, useful, reads like a report.

2. **Coaching mode** — "Your nervous system is contracting against expansion. It is a protector trying to keep you safe." This reads like a coaching session the user didn't sign up for.

**The rule:** In the report, stay in informational mode. Save coaching mode for the CTA and the Sovereign Arc itself. The report should make people feel *accurately described*, not *diagnosed*. Being described makes them curious. Being diagnosed makes them defensive.

Exceptions where coaching mode is earned:
- The Gene Key shadow→gift explanation (1 paragraph max)
- The North Node growth edge framing
- The CTA box

Everywhere else: describe, don't prescribe.
